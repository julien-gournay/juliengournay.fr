CENTRE DE CONFIDENTIALITE

Le site Julien Gournay utilise vos données personnelles pour créer la meilleure expérience possible. Découvrez-en plus sur vos droits en matière de données.

Sur mon site, je vous offre la meilleure expérience possible. Pour ce faire, je traite certaines de vos données personnelles pour comprendre vos habitudes et développer le meilleur service pour vous. Toutefois, soyez assuré que la confidentialité et la sécurité de vos données personnelles sont une priorité des plus importantes pour moi.


Le but de ce centre de confidentialité est de vous fournir plus d'informations sur les droits et les contrôles dont vous disposez sur vos données personnelles, mais aussi d'expliquer certaines sections importantes de ma Politique de confidentialité.

Quelles données personnelles je recueille sur vous ?


Il est très important pour moi que vous sachiez quelles données personnelles je recueille sur vous, comment je les recueille et pourquoi cela est nécessaire. Je collecte vos données personnelles de la manière suivante :


1. Données personnelles recueillies lorsque vous me contacter via mon site : lorsque vous me contacter à partir du formulaire de contact, je collecte certaines données personnelles afin de vous reconacter.

2. Données personnelles recueillies via votre utilisation du site : lorsque vous utilisez mon site, je collecte des données personnelles relatives à votre utilisation, comme les commentaires susceptibles d’être créés.

3. Données personnelles collectées dans le but de vous fournir des fonctions/fonctionnalités supplémentaires : vous pouvez parfois me fournir des données personnelles complémentaires ou me donner votre autorisation pour la collecte de données personnelles, par exemple pour profiter de fonctions ou fonctionnalités supplémentaires.

Pourquoi je recueille et utilise ces données personnelles ?

Je collecte et utilise vos données personnelles pour les raisons suivantes :


- Fournir, personnaliser et améliorer votre expérience sur le site et autres services par Julien Gournay.

- Comprendre comment vous accédez à mon site et comment vous l'utilisez pour garantir la fonctionnalité technique des Services, développer de nouveaux services, et analyser votre utilisation du Site.

- Communiquer avec vous pour des raisons liées a mon entreprise.

Communiquer avec vous, soit directement, soit via l'un de nos partenaires, pour : le marketing, la recherche, sondages et tirages au sort, par e-mails, notifications ou autres messages, conformément aux autorisations que vous m’avez données


Vous fournir des fonctionnalités, des informations, de la publicité ou tout autre contenu basé sur votre emplacement spécifique.


Vos droits sur vos données personnelles

Comme vous le savez peut-être, un nouveau règlement de l'Union européenne, appelé le Règlement général sur la protection des données ou « RGPD », accorde aux personnes certains droits relatifs à leurs données personnelles. En conséquence, nous avons mis en œuvre des contrôles d'accès et des mesures de transparence supplémentaires dans notre centre de confidentialité et dans nos paramètres de confidentialité, afin d'aider les utilisateurs à bénéficier de ces droits. Chaque personne dispose des droits décrits ci-dessous :

Accès à vos données personnelles

Droit d'être informé et de demander l'accès aux données personnelles que nous traitons.

Mise à jour de vos données personnelles

Droit de nous demander de modifier ou de mettre à jour vos données personnelles lorsqu'elles sont inexactes ou incomplètes.

Restriction de l'utilisation de vos données personnelles

Droit de nous demander d'arrêter temporairement ou définitivement le traitement de tout ou partie de vos données personnelles.

Refus de l'utilisation de vos données personnelles

Droit de refuser le traitement de vos données personnelles lorsque nous ne pouvons pas justifier la nécessité de ce traitement pour des raisons légitimes ou légales.

Restriction de l'utilisation de vos données personnelles

Droit de refuser le traitement de vos données personnelles à des fins de marketing direct.

Refus de l'utilisation de vos données personnelles

Droit de nous demander de modifier ou de mettre à jour vos données personnelles lorsqu'elles sont inexactes ou incomplètes.
